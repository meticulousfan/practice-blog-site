# Blog site

## How to start

- npm install
- config .env file
  ```
  DATABASE_URL="postgresql://postgres:apple@localhost:5432/postgres?schema=public"
  ```
- npx prisma migrate
- npm run dev

## How to build

- npm run build

# Technologies

- nextJs
- graphQL
- Apollo
- postgreSQL
- cypress

# How to run e2e test using cypress

- `npx cypress open`
